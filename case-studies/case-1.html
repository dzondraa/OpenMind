AI Knowledge Assistant for Operations

Context

Retail operations teams struggled to find up‑to‑date procedures across wikis and SharePoint. We built a secure AI assistant to answer workflow questions and surface relevant steps instantly.

Approach

  • RAG architecture with Azure OpenAI, embeddings, and vector search
  • Automated ingestion from SharePoint and Confluence with chunking
  • Prompt‑engineering with offline evals and safety checks
  • MLOps: CI/CD, observability, feedback loops, and guardrails

Outcomes

  • Answer time reduced from minutes to seconds
  • 40% fewer escalations to subject‑matter experts
  • Measured quality via evals; continual improvement from feedback
  • Governed data access with auditability

Tech Highlights

  • Azure OpenAI, Azure Cognitive Search/Vector DB
  • Azure Functions, Data Factory, Event Grid
  • OpenTelemetry, Application Insights, dashboards
